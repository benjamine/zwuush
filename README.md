# zwuush

## Requirements

- macOS 12+
- Swift 5.6 or later
- Xcode 13 or later

## Installation

1. **Clone the repository:**

```bash
   git clone https://github.com/benjamine/zwuush.git
   cd zwuush
   swift build --configuration release

   .build/release/zwuush

      .build/release/zwuush --sound path/to/sound.m4a

```
